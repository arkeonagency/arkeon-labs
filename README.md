# Arkeon — Studio Website

This repository contains the source code for the **Arkeon** marketing website, built with **Astro**, **React**, and **Tailwind CSS**.

## Tech Stack

- **Framework:** [Astro](https://astro.build) (v4)
- **Styling:** [Tailwind CSS](https://tailwindcss.com)
- **UI Components:** React
- **Animations:** AOS (Animate On Scroll) + Framer Motion
- **Icons:** Lucide React

## Project Structure

```bash
src/
├── components/   # Reusable UI components (Navbar, Footer, Hero...)
├── layouts/      # Base layouts (BaseLayout.jsx)
├── pages/        # Astro pages (routes)
│   ├── work/     # Portfolio case study pages
│   └── ...
├── styles/       # Global CSS
└── content/      # (Optional) Markdown content
